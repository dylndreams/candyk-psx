## Installation

### Binary

1. Install or compile pacman (the Arch Linux package manager).
2. Append the following to /etc/pacman.conf:

```
[candyk]
Server = http://candyk.asie.pl/repo/x86_64
```

3. `pacman -Syu`
4. `pacman -S candyk-psx`
